# **App Name**: AWS Cloud Club – MRIT

## Core Features:

- Dynamic Content Management & Display: Fetch and display events, team members, and projects from Firebase Firestore using responsive card layouts with loading skeletons.
- Static Information Pages: Render essential static pages (Home, About, Join) with hero sections, call-to-actions, and content previews. The Home page includes sections for about, what we do, stats, and featured events.
- Responsive Navigation System: Implement a mobile-first navigation bar with a hamburger menu for seamless browsing, and a persistent footer with social links and contact information.
- Cloud Project Idea Tool: An AI tool that generates tailored project titles, descriptions, tech stacks, and implementation steps based on user-selected AWS services and interest areas using Genkit and Gemini API.
- External Join Form Integration: Integrate a 'Join Club' call-to-action button that redirects users to an external Google Form for registration.
- Firebase Hosting Deployment: Deploy the production-ready application to Firebase Hosting, ensuring optimized performance and responsiveness.

## Style Guidelines:

- Primary theme: AWS Pro Dark UI. Background: A dark charcoal blue (#192126).
- Accent colors: Vibrant orange (#EB7E1C) and a modern blue (#3399CC) for highlights, CTAs, and key elements, inspired by AWS branding.
- Neon accents: The 'AI Cloud Project Generator' section will feature vibrant, glowing neon-like colors (e.g., electric blue, bright green) to signify its generative nature and create a dynamic, modern feel.
- Font family: 'Inter' (sans-serif) for clean, modern, and highly readable text across all UI elements, reflecting a professional and startup vibe.
- Utilize 'Heroicons' (outlined style) for a consistent, professional, and clean visual language throughout the application.
- Fully responsive grid system for all sections, ensuring optimal display on mobile and desktop. Cards will feature a 'glassmorphism' effect with frosted transparency, subtle blurring, and rounded corners, providing depth and a contemporary look.
- Smooth hover effects for interactive elements, gentle section fade-in transitions for content loading, and engaging skeleton loaders for asynchronous data fetching.
- Smooth scrolling across all sections to enhance user experience, complementing the modern and professional aesthetic.
- Subtle glow effects for neon accents within the AI section to enhance its dynamic and unique presentation.
