# 🚀 AWS Cloud Club – MRIT

Official website for the **AWS Cloud Club at Mysuru Royal Institute of Technology (MRIT)**.

🌐 **Live Website:** https://aws-cloud-club-mrit.vercel.app

---

## 📌 About the Project

This is a modern, responsive web platform built to:

* Showcase AWS Cloud Club activities
* Manage events and announcements
* Display team members and projects
* Help students explore cloud technologies

---

## ✨ Features

* 🎯 Dynamic event display
* 👥 Team showcase section
* 🧠 AI-powered cloud project idea generator
* 📱 Fully responsive UI

---

## 🛠️ Tech Stack

* **Frontend:** Next.js, TypeScript, Tailwind CSS
* **Backend:** Firebase (Firestore, Hosting)
* **Deployment:** Vercel
* **AI Integration:** Gemini / Genkit

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/zubaida-yasmeen/aws-cloud-club-mrit.git
cd aws-cloud-club-mrit
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run locally

```bash
npm run dev
```
